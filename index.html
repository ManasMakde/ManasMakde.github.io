Calendar App
JavaScript, React Native
A simple reusable, scalable, memoized calender widget A calender is a well known widget and is often used in various mobile applications but there isn’t a viable module for react native Based apps. Usually the existing modules are either too large in size with bloated and redundant features or they are outdated and hard to integrate with in an existing app This project aims to solve that by introducing a simple calendar, optimized with memoization and ready to use out of the box with little to no setup and cross compatible across all versions with a minimal size of ~26KB making it extremely light weight
